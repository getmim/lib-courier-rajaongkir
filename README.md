# lib-courier-rajaongkir

## Instalasi

Jalankan perintah di bawah di folder aplikasi:

```
mim app install lib-courier-rajaongkir
```

## Konfigurasi

Tambahkan konfigurasi seperti di bawah pada konfigurasi apliasi:

```php
return [
    'libCourierRajaOngkir' => [
        'package' => 'starter',
        'apikey' => ''
    ]
];
```
